# 식각공정 매뉴얼 (Etch Process Manual)

**문서번호**: SKH-ETCH-MAN-001  
**개정이력**: Rev. 3.2 (2025-03-10)  
**작성부서**: 식각공정팀  
**적용라인**: FAB-M16 / FAB-M17 / FAB-C1

---

## 1. 개요

식각공정(Etch Process)은 반도체 제조에서 포토리소그래피로 정의된 패턴에 따라 불필요한 박막 또는 기판 물질을 선택적으로 제거하는 핵심 단위공정이다. 본 매뉴얼은 건식 식각(Dry Etch) 및 습식 식각(Wet Etch) 양 방식의 원리, 공정 파라미터, 관리 기준 및 주요 불량 유형을 정의한다.

---

## 2. 식각 방식별 원리

### 2.1 건식 식각 (Dry Etch)

건식 식각은 플라즈마(Plasma) 환경에서 반응성 이온과 라디칼을 이용하여 박막을 제거하는 방식이다.

**주요 메커니즘:**

| 메커니즘 | 원리 | 특징 |
|---------|------|------|
| 화학적 식각 (Chemical) | 라디칼과 기판 물질의 화학반응 | 등방성, 선택비 높음 |
| 물리적 식각 (Physical/Sputtering) | 이온의 운동에너지에 의한 기계적 제거 | 이방성, 선택비 낮음 |
| 반응성 이온 식각 (RIE) | 화학 + 물리 복합 메커니즘 | 이방성 + 선택비 양립 |
| 고밀도 플라즈마 (HDP) | 고이온밀도 환경, 낮은 압력 | 고종횡비 구조에 적합 |

**사용 장비:** LAM Versys, AMAT Centris, TEL Tactras

**대표 식각 반응 (SiO₂ 건식 식각):**
```
SiO₂ + CF₄ → SiF₄(g) ↑ + CO₂(g) ↑
```

### 2.2 습식 식각 (Wet Etch)

습식 식각은 액상 화학약품(Chemical Solution)을 이용하여 박막을 제거하는 방식으로, 주로 세정(Cleaning) 또는 등방성 제거가 필요한 단계에 사용된다.

**주요 식각액 종류:**

| 대상 물질 | 식각액 | 비율 | 온도 |
|---------|------|------|------|
| SiO₂ | BOE (HF:NH₄F) | 1:6 | 상온 |
| Si₃N₄ | 열 인산 (H₃PO₄) | 85% | 160°C |
| Al | H₃PO₄:HNO₃:CH₃COOH:H₂O | 16:1:1:2 | 40°C |
| 열산화막 | HF 희석액 | 1:100 | 상온 |

**특징:**
- 등방성 식각으로 언더컷(Undercut) 발생 가능
- 배치(Batch) 처리 가능, 처리량(Throughput) 우수
- 미세 패턴(45nm 이하)에는 부적합

## 3. 공정 파라미터 (Process Parameters)

### 3.1 건식 식각 주요 파라미터

| 파라미터 | 범위 | 단위 | 영향 인자 |
|---------|------|------|---------|
| RF Power (Source) | 200 ~ 2,000 | W | 플라즈마 밀도, 이온 에너지 |
| RF Power (Bias) | 0 ~ 500 | W | 이온 방향성, 이방성 |
| Chamber Pressure | 2 ~ 100 | mTorr | 평균자유행로, 이온-라디칼 비율 |
| Gas Flow (주 식각 가스) | 50 ~ 500 | sccm | 반응 속도, 균일도 |
| Gas Flow (희석/패시베이션) | 10 ~ 200 | sccm | 선택비, 폴리머 형성 |
| 기판 온도 (Chuck Temp) | -20 ~ 80 | °C | 반응속도, 폴리머 증착률 |
| 식각 시간 | 공정별 상이 | sec | 목표 CD, 잔류막 두께 |

### 3.2 관리 기준값 (Control Limits)

```
RF Power 편차:    ±5% (경보 기준: ±8%)
압력 편차:        ±3 mTorr (경보 기준: ±5 mTorr)
Gas Flow 편차:    ±2% (경보 기준: ±5%)
식각률 편차:      ±3% (웨이퍼 내 균일도 기준)
CD 편차:          ±5 nm (설계 CD 기준)
```

### 3.3 엔드포인트 검출 (Endpoint Detection)

엔드포인트는 식각 완료 시점을 자동 검출하는 기능으로, 과식각(Over-etch)을 방지한다.

- **OES (Optical Emission Spectroscopy)**: 플라즈마 발광 스펙트럼 변화로 검출
- **Laser Interferometry**: 반사광 간섭 패턴으로 막 두께 실시간 측정
- **Mass Spectrometry**: 반응 부산물 농도 변화 모니터링

## 4. 주요 불량 유형 및 관리

### 4.1 불량 분류

| 불량 유형 | 원인 | 영향 | 조치 방법 |
|---------|------|------|---------|
| 과식각 (Over-etch) | 엔드포인트 오검출, 식각률 증가 | CD 증가, 하부막 손상 | 엔드포인트 파라미터 재조정 |
| 불완전 식각 (Under-etch) | 식각률 감소, 마스크 손상 | 패턴 잔류물 발생 | 식각 시간 연장, 가스 조성 변경 |
| 언더컷 (Undercut) | 등방성 식각 과다 | Bias 편차 조정, 압력 감소 | 패턴 폭 감소 |
| 마이크로로딩 (Micro-loading) | 패턴 밀도 차이로 인한 식각률 편차 | CD 균일도 저하 | 가스 조성 및 압력 최적화 |
| 노칭 (Notching) | 이온 산란에 의한 하부 식각 | 소자 특성 열화 | Bias power 감소 |
| 잔류 폴리머 (Polymer Residue) | 탄소계 폴리머 과도 생성 | 세정 불량, 후속공정 불량 | O₂ 애싱(Ashing) 추가 |

### 4.2 Particle 불량 관리

식각 챔버 내 Particle은 수율에 직접적인 영향을 미치므로 특별 관리가 필요하다.

- **발생 원인**: 챔버 내벽 폴리머 박리, 세라믹 파편, 전극 마모
- **관리 주기**: 인시추(In-situ) 세정 — 매 50 RF-hour마다 수행
- **모니터링**: KLA-Tencor SP7 또는 동급 파티클 검사 장비로 매 Lot 검사
- **허용 기준**: ≥0.12μm 파티클 10개/웨이퍼 이하 (Critical Layer 기준)

## 5. 안전 및 환경 관리

### 5.1 사용 화학물질 위험 등급

| 화학물질 | GHS 분류 | 특이사항 |
|---------|---------|---------|
| CF₄ | 온실가스 | 지구온난화지수(GWP) 7,380 (IPCC AR5, 100년 기준) |
| HF (불화수소) | 급성독성, 부식성 | 피부 침투, 특별 취급 필수 |
| Cl₂ | 급성독성, 부식성 | 누출 시 즉시 대피 |
| BCl₃ | 부식성, 가수분해성 | 수분 접촉 금지 |

### 5.2 비상 조치 절차

1. 가스 누출 감지 시 → 즉시 공정 중단 → 대피 → 안전팀 연락 (내선: 119)
2. HF 피부 접촉 시 → 즉시 다량의 물로 세척(15분) → 의무실 보고
3. 챔버 열기 전 → 반드시 퍼지(Purge) 완료 확인 → O₂ 농도 측정 후 진입
