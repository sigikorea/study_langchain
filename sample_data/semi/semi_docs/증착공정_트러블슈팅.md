# 증착공정 트러블슈팅 가이드 (Deposition Process Troubleshooting)

**문서번호**: SKH-DEP-TRB-002  
**개정이력**: Rev. 2.5 (2025-02-20)  
**작성부서**: 증착공정팀  
**적용장비**: AMAT Producer (CVD), AMAT Endura (PVD), Novellus Concept (CVD)

---

## 1. 개요

본 트러블슈팅 가이드는 증착공정(CVD/PVD) 운영 중 발생하는 주요 이상 현상에 대해 원인을 분석하고 체계적인 대응 절차를 제공한다. 현장 엔지니어는 본 문서의 Decision Tree를 따라 이상 현상을 진단하고 신속하게 조치한다.

---

## 2. 증착공정 원리

### 2.1 화학기상증착 (CVD, Chemical Vapor Deposition)

전구체(Precursor) 가스가 가열된 기판 표면에서 화학반응을 일으켜 박막을 형성한다.

**CVD 유형별 특성:**

| 방식 | 온도 범위 | 압력 | 주요 용도 |
|------|---------|------|---------|
| APCVD | 350~500°C | 대기압 | BPSG, PSG 산화막 |
| LPCVD | 550~750°C | 0.1~2 Torr | 폴리실리콘, Si₃N₄, SiO₂ |
| PECVD | 250~400°C | 100~500 mTorr | 층간절연막(ILD), SiN 보호막 |
| ALD | 100~400°C | 0.1~5 Torr | High-k 게이트 유전체, TiN 배리어 |
| HDP-CVD | 250~400°C | 1~10 mTorr | 갭필(Gap-fill) 산화막 |

**SiO₂ PECVD 반응식:**
```
SiH₄ + 2N₂O → SiO₂ + 2N₂ + 2H₂ (플라즈마 보조)
```

### 2.2 물리기상증착 (PVD, Physical Vapor Deposition)

타겟(Target) 물질을 물리적으로 기화시켜 기판에 증착한다. 주로 금속 배선(Al, Ti, TiN, Cu Seed) 형성에 사용된다.

**스퍼터링 반응:**
```
Ar⁺ + Target → Target원자 + 2차전자
Target원자 → 기판 표면 증착
```

## 3. 트러블슈팅 Decision Tree

### 3.1 박막 두께 이상 (Thickness Deviation)

```
박막 두께 이상 발생
├── 두께 부족 (Under-deposition)
│   ├── 전구체 공급 확인
│   │   ├── 가스 유량 정상 → 챔버 온도 확인
│   │   │   ├── 온도 정상 → 전구체 분해율 확인 → ALD 사이클 수 점검
│   │   │   └── 온도 이상 → 히터 교체 또는 PID 재조정
│   │   └── 가스 유량 이상 → MFC 교체 또는 재보정
│   └── 전구체 공급 정상이나 두께 부족
│       ├── 베이크아웃(Bakeout) 미완료 → 챔버 컨디셔닝 수행
│       └── 챔버 컨디션 불량 → 기준 웨이퍼로 시즈닝(Seasoning)
│
└── 두께 초과 (Over-deposition)
    ├── 증착 시간 오류 → 레시피 재확인 및 수정
    ├── 증착률(Deposition Rate) 상승
    │   ├── 챔버 압력 상승 → 배기 밸브 점검, 펌프 성능 확인
    │   └── 가스 유량 증가 → MFC 재보정
    └── 엔드포인트 오동작 → 엔드포인트 센서 점검
```

### 3.2 막질 이상 (Film Quality Deviation)

```
막질 이상 발생
├── 굴절률(Refractive Index) 편차
│   ├── H/Si 비율 변화 → 수소 함량 TDS로 분석 → 공정온도/RF Power 조정
│   └── 탄소 오염 → 전구체 순도 확인, 라인 퍼지 수행
│
├── 잔류응력(Residual Stress) 이상
│   ├── 인장응력 과다 → 증착온도 증가, N₂ 유량 감소
│   └── 압축응력 과다 → RF Bias power 감소, 증착속도 증가
│
└── 전기적 특성 이상
    ├── 누설전류 증가 → 막 내 결함 TEM 분석 → 공정 재최적화
    └── 절연파괴 전압 저하 → 핀홀(Pinhole) 검사 → 두께 증가 검토
```

### 3.3 Particle 불량 발생

```
Particle 불량 발생 (기준 초과: ≥0.12μm, 10개/웨이퍼 초과)
├── [즉시 조치] 공정 홀드 → QE 통보 → 원인 조사 착수
│
├── Particle 위치 분석 (KLA SP7 맵 확인)
│   ├── 웨이퍼 엣지 집중 → 척(Chuck) 세정 또는 교체
│   ├── 웨이퍼 중앙 집중 → 샤워헤드(Showerhead) 점검 및 세정
│   ├── 전체 분산 → 챔버 내벽 박리 의심 → 챔버 세정(Wet clean) 수행
│   └── 특정 패턴 → 가스 라인 오염 → 라인 퍼지 및 필터 교체
│
├── 챔버 상태 점검
│   ├── 챔버 세정 이력 확인 (마지막 PM 이후 RF-hour 확인)
│   │   ├── PM 주기 초과 → 즉시 PM 수행
│   │   └── PM 주기 내 → 비정상 Particle 원인 추가 조사
│   └── 이전 공정 잔류물 확인 → 인시추(In-situ) 세정 수행
│
└── 복구 절차
    ├── 챔버 세정 수행 (NF₃ Remote Plasma Clean)
    ├── 기준 웨이퍼 3매로 모니터링 공정 진행
    ├── Particle 기준치 이하 확인 후 생산 재개
    └── 원인 및 조치 내용 MES에 기록
```

## 5. 예방정비 (PM) 체크리스트

### 5.1 월간 PM (CVD 기준)

- [ ] 챔버 내벽 육안 검사 및 박리 여부 확인
- [ ] 샤워헤드 세정 (희석 HF 처리 후 DI 린스)
- [ ] 히터 저항값 측정 및 균일도 확인 (편차 <±3%)
- [ ] 진공 펌프 오일 레벨 확인 및 교환 (필요시)
- [ ] 가스 라인 필터 차압 확인
- [ ] MFC 가스 유량 재보정 (N₂ 기준)
- [ ] 리크 테스트 수행 (기준: <1×10⁻⁸ Torr·L/s)

### 5.2 분기 PM (PVD 기준)

- [ ] 타겟 두께 측정 및 잔여 수명 계산
- [ ] 차폐판(Shield) 교체
- [ ] 냉각수 배관 세정
- [ ] 기판 반송계 청소 및 윤활
- [ ] 진공 게이지 교정

---

## 6. 에스컬레이션 기준

아래 상황 발생 시 즉시 공정 팀장 및 장비 엔지니어에게 보고한다.

1. 동일 알람 1시간 이내 3회 이상 반복 발생
2. Particle 기준 초과 후 챔버 세정에도 불구하고 재발
3. 박막 두께 편차 ±10% 초과
4. 장비 비가동 시간 예상 4시간 초과
5. 신규 불량 모드 발생 (기존 분류에 해당 없는 경우)
