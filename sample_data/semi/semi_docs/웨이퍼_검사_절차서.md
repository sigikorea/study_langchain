# 웨이퍼 검사 절차서 (Wafer Inspection Procedure)

**문서번호**: SKH-QC-INS-005  
**개정이력**: Rev. 3.3 (2025-03-25)  
**작성부서**: 품질보증팀 (QA)  
**적용범위**: FAB-M16 / FAB-M17 / FAB-C1 전 공정

---

## 1. 목적 및 범위

본 절차서는 SK하이닉스 반도체 제조 라인에서 수행하는 웨이퍼 검사의 종류, 방법, 판정 기준, 후속 조치를 규정한다. 검사 담당자는 본 절차서에 따라 검사를 수행하고 결과를 MES에 정확하게 기록하여야 한다.

검사의 목적:
- 공정 이상을 조기 발견하여 수율 손실을 최소화
- 불량 웨이퍼의 후속 공정 투입 방지
- 공정 능력(Cp/Cpk) 데이터 축적 및 트렌드 관리

---

## 2. 검사 종류

### 2.1 육안 검사 (Visual Inspection)

| 검사 항목 | 방법 | 판정 기준 |
|---------|------|---------|
| 웨이퍼 표면 오염 | 100룩스 조명 하 45° 각도 확인 | 육안으로 식별 가능한 오염 없음 |
| 스크래치 | 고휘도 조명 반사 확인 | 스크래치 없음 (길이 1mm 초과 불허) |
| 칩핑(Chipping) | 웨이퍼 엣지 확인 | 0.5mm 초과 칩핑 없음 |
| 박막 균일도 | 반사광 색상 분포 확인 | 색상 불균일 영역 없음 (육안 판단) |
| 포토레지스트 코팅 | 반사광 균일도 | 코팅 누락, 기포 없음 |

**육안 검사 수행 조건:**
- 클린룸 내 지정 검사대에서 수행
- 반드시 클린 글러브 착용
- 웨이퍼 캐리어(FOUP) 외부에서 카세트 상태 먼저 확인
- 검사 중 웨이퍼 표면에 직접 접촉 금지 (핀셋 또는 진공 픽업 사용)

### 2.2 광학 검사 (Optical Defect Inspection)

**검사 장비 및 적용 기준:**

| 장비 | 검출 하한 | 검사 모드 | 적용 Layer |
|------|---------|---------|---------|
| KLA SP7 | 19 nm | Dark Field / Bright Field | Critical Layer 전수 |
| KLA Puma 9850 | 65 nm | Dark Field | Non-Critical 샘플링 |
| Rudolph NSX-320 | 120 nm | Dark Field | 공정 모니터링 |

**검사 절차:**

```
① 웨이퍼 식별 → FOUP에서 웨이퍼 정보 스캔 (바코드 또는 OCR)
② 검사 레시피 선택 → Layer 코드에 매핑된 레시피 자동 로딩
③ 장비 자동 검사 수행 → 약 1~3분/웨이퍼
④ 결과 자동 업로드 → KLARF 파일 → KLA Klarity 서버
⑤ 결과 판정 → MES 자동 판정 (기준치 초과 시 홀드 발생)
⑥ 이상 시 → 담당 엔지니어 확인 후 추가 분석 진행
```

**샘플링 정책:**

| Layer 분류 | 샘플링 비율 | 근거 |
|---------|---------|------|
| Critical (Gate, Contact, M1) | 100% | 불량 영향 최대 |
| Semi-Critical (M2~M4) | 50% | 중간 영향 |
| Non-Critical (M5 이상, 후공정) | 25% | 최소 영향 |
| PM 후 복귀 Lot | 100% | PM 후 컨디션 확인 필수 |

### 2.3 전기적 검사 (Electrical Test)

전기적 검사는 웨이퍼 전체에 대해 다이(Die) 단위로 전기 특성을 측정하며, 제조 완료 후 최종 수율을 결정한다.

**검사 단계:**

| 단계 | 명칭 | 시점 | 목적 |
|------|------|------|------|
| E-test (공정 중) | PCM (Process Control Monitor) | 주요 공정 완료 후 | 공정 파라미터 확인 |
| Probe Test (EDS) | Electrical Die Sort | 전공정 완료 후 | 다이 양부 판정 |
| Final Test | FT | 패키징 후 | 최종 전기 특성 |

**PCM (Process Control Monitor) 측정 항목:**

| 측정 항목 | 기호 | 목표 범위 | 이상 기준 |
|---------|------|---------|---------|
| 게이트 산화막 두께 | Tox | 목표±3Å | ±5Å 초과 |
| NMOS 문턱전압 | Vtn | 목표±30mV | ±50mV 초과 |
| PMOS 문턱전압 | Vtp | 목표±30mV | ±50mV 초과 |
| 접합 누설전류 | Ij | <1 pA/μm | >5 pA/μm |
| 게이트 누설전류 | Ig | <0.1 nA/μm² | >1 nA/μm² |
| 콘택 저항 | Rc | <10 Ω | >20 Ω |
| 배선 저항 | Rs | 목표±10% | ±20% 초과 |

**EDS (Electrical Die Sort) 결과 분류:**

| 코드 | 의미 | 처리 방법 |
|------|------|---------|
| 1 (Good) | 모든 전기 특성 합격 | 후공정(패키징) 진행 |
| 0 (Fail) | 1개 이상 특성 불합격 | 스크랩 또는 리워크 검토 |
| 2 (Marginal) | 특성 경계선 상 | 최종 테스트 후 판정 |
| I (Ink) | 불량 마킹됨 | 스크랩 |

### 2.4 막 두께 / CD 측정

| 측정 항목 | 장비 | 방법 | 주기 |
|---------|------|------|------|
| 박막 두께 | Nova NovaScan, Ellipsometer | 광학적 모델 피팅 | 공정 후 샘플링 |
| CD (Critical Dimension) | KLA-Tencor CD-SEM | SEM 측정 | 리소그래피 후 전수 |
| 오버레이 | KLA-Tencor Archer 500 | 광학 정렬 마크 | 리소그래피 후 전수 |
| 표면 거칠기 | AFM | 원자력 현미경 | 연구 목적 및 이상시 |

## 3. 합격/불합격 판정 기준

### 3.1 광학 검사 판정

| 구분 | 판정 기준 | 조치 |
|------|---------|------|
| 합격 (PASS) | 총 결함 수 ≤ 기준치 AND Critical 결함 0개 | 후속 공정 진행 |
| 조건부 합격 | 총 결함 수 ≤ 기준치 × 1.5배 AND Critical 결함 0개 | 엔지니어 확인 후 진행 |
| 불합격 (FAIL) | 총 결함 수 > 기준치 × 1.5배 OR Critical 결함 ≥1개 | MES 홀드 + QE 보고 |

**Layer별 Particle 허용 기준 (≥0.12μm 기준):**

| Layer 분류 | 허용 기준 (개/웨이퍼) | 조건부 합격 |
|---------|------------------|---------|
| Gate / Active | ≤ 5 | 6 ~ 7 |
| Contact / Via | ≤ 8 | 9 ~ 12 |
| Metal 1 (M1) | ≤ 10 | 11 ~ 15 |
| Metal 2 이상 | ≤ 20 | 21 ~ 30 |

### 3.2 전기적 검사 합격 기준

| 검사 | 합격 기준 | 비고 |
|------|---------|------|
| PCM | 전 항목 관리 범위 이내 | 1개 항목 이상 이탈 시 로트 홀드 |
| EDS 수율 | ≥ 공정별 목표 수율 | 제품별 상이 (MES 레시피 참조) |
| EDS 수율 하한 | 목표 수율 × 0.85 | 이하 시 QE 보고 및 원인 분석 필수 |

## 4. 리워크 (Rework) 절차

### 4.1 리워크 가능 여부 판단

| 불량 단계 | 리워크 가능 여부 | 리워크 방법 |
|---------|-------------|---------|
| 포토레지스트 도포/노광 불량 | 가능 | 포토레지스트 제거 후 재도포 |
| 현상 불량 | 가능 (노광 전) | 포토레지스트 제거 후 재도포 |
| 식각 불량 (과식각) | 불가 | 스크랩 |
| 식각 불량 (불완전 식각) | 가능 | 추가 식각 수행 |
| CMP 불량 (두께 초과) | 가능 | 추가 연마 수행 |
| CMP 불량 (두께 부족) | 불가 | 스크랩 |
| Particle 오염 (표면) | 가능 | 세정 공정 수행 |
| 금속 오염 | 조건부 (오염 수준 따라) | 세정 또는 스크랩 |

### 4.2 리워크 이력 관리

- 리워크 수행 횟수: 동일 층 최대 2회 (3회 이상 시 스크랩 처리)
- MES에 리워크 이력 반드시 기록 (리워크 코드: RWK-YYYYMMDD-NNN)
- 리워크 Lot은 후속 공정에서 추적 검사 대상으로 지정

## 5. 검사 기록 및 보고

### 5.1 MES 기록 항목

검사 완료 후 반드시 MES에 아래 항목을 기록한다:

- Lot ID, 웨이퍼 번호
- 검사 장비 ID, 검사 레시피명
- 검사 날짜/시간
- 결함 수 (크기별 분류)
- 판정 결과 (Pass / Fail / Conditional)
- 특이사항 (이상 결함 형상 등)
- 담당자 사번 및 서명

### 5.2 정기 보고

| 보고 종류 | 주기 | 수신자 | 내용 |
|---------|------|------|------|
| 일간 검사 현황 | 매일 오전 9시 | 공정팀장, QE | Lot별 검사 결과 요약 |
| 주간 수율 보고 | 매주 월요일 | 부서장 | 수율 트렌드, 이슈 현황 |
| 월간 품질 보고 | 매월 첫째 주 | 임원진 | 수율, 불량 분석, 개선 계획 |
| 이상 발생 즉시 보고 | 발생 즉시 | 팀장 → 부서장 | 불량 현황 및 조치 계획 |
