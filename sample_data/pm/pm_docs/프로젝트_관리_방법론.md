# 프로젝트 관리 방법론

## 개요

프로젝트 관리 방법론은 프로젝트의 시작부터 종료까지 체계적으로 계획·실행·통제하기 위한 표준 프레임워크입니다. 본 문서는 SK하이닉스 내부 프로젝트에 적용되는 핵심 방법론을 정의합니다.

---

## 1. 프로젝트 생애주기 (Project Lifecycle)

프로젝트는 아래 5단계 생애주기를 따릅니다.

| 단계 | 주요 활동 | 산출물 |
|------|-----------|--------|
| 착수 (Initiation) | 요구사항 수집, 이해관계자 식별 | 프로젝트 헌장 (Project Charter) |
| 계획 (Planning) | WBS 작성, 일정·예산 수립 | 프로젝트 관리 계획서 |
| 실행 (Execution) | 개발·구현, 팀 관리 | 산출물 (코드, 문서, 보고서) |
| 통제 (Monitoring & Control) | 진척 모니터링, 변경 관리 | 현황 보고서, 이슈 로그 |
| 종료 (Closing) | 인수인계, 교훈 기록 | 프로젝트 종료 보고서 |

---

## 2. WBS (Work Breakdown Structure)

WBS는 프로젝트 범위를 계층적으로 분해하여 관리 가능한 작업 단위(Work Package)로 나누는 기법입니다.

### 작성 원칙

- **100% 규칙**: WBS는 프로젝트 범위 전체를 포함해야 합니다.
- **상호 배타성**: 동일한 작업이 두 Work Package에 중복 포함되어서는 안 됩니다.
- **분해 깊이**: 작업 기간이 8~80시간 수준이 되도록 분해합니다.

### WBS 코드 체계 예시

```
1.0 ERP 시스템 고도화
  1.1 요구사항 분석
    1.1.1 현행 시스템 분석
    1.1.2 사용자 인터뷰
  1.2 설계
    1.2.1 DB 스키마 설계
    1.2.2 API 설계
  1.3 개발
  1.4 테스트
  1.5 배포 및 안정화
```

## 3. 간트 차트 (Gantt Chart)

간트 차트는 프로젝트 일정을 시각적으로 표현하는 막대형 차트입니다.

### 구성 요소

- **작업 목록**: WBS 기반 작업 항목
- **타임라인**: 날짜 축 (주 또는 월 단위)
- **의존 관계**: 선행 작업(Predecessor) 연결선
- **마일스톤**: 주요 완료 시점 (다이아몬드 기호 ◆)
- **기준선(Baseline)**: 최초 계획 대비 현재 진척 비교

---

## 4. RACI 매트릭스

RACI는 프로젝트 역할과 책임을 명확히 정의하는 도구입니다.

| 기호 | 의미 | 설명 |
|------|------|------|
| R | Responsible | 실제 작업을 수행하는 담당자 |
| A | Accountable | 최종 책임자 (1인만 지정) |
| C | Consulted | 의견을 제공하는 전문가 |
| I | Informed | 결과를 통보받는 이해관계자 |

### RACI 예시

| 작업 | PM | 개발자 | QA | 임원 |
|------|----|---------|----|------|
| 요구사항 확정 | A | C | C | I |
| 개발 구현 | I | R | C | - |
| 테스트 승인 | A | C | R | I |
| 배포 결정 | R | C | I | A |

---

## 5. EVM (Earned Value Management)

EVM은 계획 대비 실제 진척과 비용을 동시에 측정하는 성과 관리 기법입니다.

### 핵심 지표

- **PV (Planned Value)**: 현 시점까지 계획된 작업의 예산 가치
- **EV (Earned Value)**: 현 시점까지 완료된 작업의 예산 가치
- **AC (Actual Cost)**: 현 시점까지 실제 투입된 비용

### 성과 지수

| 지수 | 공식 | 의미 |
|------|------|------|
| SPI (일정 성과 지수) | EV / PV | 1.0 이상: 일정 준수, 1.0 미만: 지연 |
| CPI (비용 성과 지수) | EV / AC | 1.0 이상: 예산 절감, 1.0 미만: 초과 |
| SV (일정 차이) | EV - PV | 양수: 선행, 음수: 지연 |
| CV (비용 차이) | EV - AC | 양수: 절감, 음수: 초과 |

---

## 6. 적용 기준

- 예산 1억 원 이상 또는 기간 3개월 이상 프로젝트: WBS + 간트 차트 + EVM 필수 적용
- 예산 5천만 원 미만 소규모 프로젝트: 간소화된 작업 목록 및 주간 현황 보고로 대체 가능
- 모든 프로젝트: RACI 매트릭스 작성 필수
